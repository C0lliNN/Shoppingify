# Shoppingify

![Shoppingify](https://i.imgur.com/shBchz3.png)

A Shop List Web Application develop in the MERN Stack

[Demo](#)

## Features

* Add Shopping Items
* Remove Shopping Items
* Create Lists
* Mark List as completed or canceled
* See Shopping List History
* See Statistics
* Authentication
